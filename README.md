# squnk-driver-mysql

## Install

```sh
  npm install AndyA/squnk-driver-mysql
```

## Usage

If your squnk.json refers to a mysql:// database URI Squnk will attempt to load this
module automatically.

```json
{
   "baseDir" : "squnk",
   "connection" : "mysql://root@localhost/test",
   "plugins" : {}
}
```
